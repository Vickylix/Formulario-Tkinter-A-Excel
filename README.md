# Formulario donde se guardan los datos en un archivo excel usando python y openpyxl
# dato importante el programa fue hecho en python 3 version (3.14.3)
##¿que hace?
Aplicación de escritorio desarrollada en Python utilizando Tkinter, Permite al usuario ingresar datos mediante un formulario y almacenarlos en un archivo Excel utilizando openpyxl.

##¿Que funciones tiene?
  * Es obligatorio rellenar cada campo.
  * Verificacion de que el Email introducido exista.
  * Verificacion de datos numeros sea enteros y no decimales. ( Edad y Telefono )
  * Creacion automatica de un archivo Excel (en caso de que se la primera ejecucion o ya exista un archivo excel con el mismo nombre.
  * Guardado dinamico de datos en el archivo.
  * Interfaz con colores oscuros para no dañar la vista.
## ¿Que tegnologias use?
  * Python 3
  * Tkinter
  * Openpyxl
  * Manejo de archivos con os
  * Expresion reguladores (re)

## Cómo ejecutar el proyecto

  1. Clonar el repositorio o descargar el repositorio
  2. Instalar extension:
    * pip install openpyxl
  3. Ejecutar el archivo principal:
     
   python "Formulario_tkinter".py
